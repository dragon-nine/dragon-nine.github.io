직장인 잔혹사 — 개인정보 처리방침

시행일: 2026년 3월 15일 | 최종 수정일: 2026년 3월 15일

DragonNine Studio(이하 "회사")는 「직장인 잔혹사」(이하 "게임") 이용자의 개인정보를 중요하게 여기며, 개인정보 보호법 및 관련 법령을 준수합니다. 본 방침은 게임에서 어떤 정보를 수집·이용·보관하는지 안내합니다.

1. 수집하는 개인정보 항목

본 게임은 회원가입, 로그인, 결제 기능이 없으며, 이용자를 직접 식별할 수 있는 개인정보를 수집하지 않습니다.

다만, 다음과 같은 비식별 정보가 자동으로 처리될 수 있습니다:

  • 게임 설정 정보: 배경음악·효과음 음소거 여부 (기기 내 localStorage에만 저장, 외부 전송 없음)
  • 게임 플레이 통계: 게임 점수, 플레이 이벤트 (게임 시작·종료·부활 등) — 토스 인앱 환경에서만 토스 서버로 전송
  • 리더보드 점수: 게임 종료 시 점수가 토스 Game Center 리더보드에 제출될 수 있음 — 토스 인앱 환경에서만 해당

2. 개인정보의 수집·이용 목적

  • 게임 설정 저장 (음소거 상태 유지)
  • 게임 플레이 통계 분석 및 서비스 개선 (토스 인앱 환경 한정)
  • 리더보드 순위 제공 (토스 인앱 환경 한정)

3. 개인정보의 보유 및 이용 기간

  • 게임 설정 (localStorage): 이용자가 브라우저 데이터를 삭제하거나 앱을 제거할 때까지 기기 내에 보관
  • 토스 플레이 통계·리더보드: 토스의 개인정보 처리방침에 따름

4. 개인정보의 제3자 제공

본 게임은 이용자의 개인정보를 제3자에게 제공하지 않습니다.

단, 토스 인앱 환경에서 실행 시 게임 점수 및 플레이 통계가 (주)비바리퍼블리카(토스)에 전송될 수 있으며, 이는 토스 플랫폼의 개인정보 처리방침에 따릅니다.

5. 개인정보의 파기

  • localStorage 데이터: 브라우저 설정에서 사이트 데이터 삭제 시 즉시 파기
  • 토스 관련 데이터: 토스 앱 삭제 또는 토스 고객센터를 통해 삭제 요청 가능

6. 이용자의 권리

이용자는 언제든지 다음과 같은 권리를 행사할 수 있습니다:

  • 게임 설정 데이터 삭제: 브라우저에서 사이트 데이터 삭제
  • 게임 이용 중단: 앱 삭제 또는 웹사이트 방문 중단
  • 개인정보 관련 문의: 아래 연락처로 문의

7. 자동 수집 장치의 설치·운영 및 거부

본 게임은 쿠키(Cookie)를 사용하지 않습니다. localStorage를 사용하여 음소거 설정만 저장하며, 이는 이용자의 기기에서만 처리됩니다.

8. 아동의 개인정보 보호

본 게임은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 아동이 개인정보를 제공한 사실을 알게 된 경우 즉시 해당 정보를 삭제합니다.
